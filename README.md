# CSV Splitter

This script splits a large CSV file into multiple smaller xlsx files.

The maximum record limit for a single xlsx worksheet is 1,048,576 rows.

### Usage

Install dependencies using `pdm`, a python dependency manager.

```bash
pdm install
```

Name the input csv file `large_input.csv`.

Run it.

```bash
python main.py
```
